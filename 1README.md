# SENSEİ CRM

## Kurulum (ilk kez)

1. Bu klasörü masaüstüne çıkar
2. Klasörün içinde sağ tıkla → "PowerShell'de Aç" (veya Terminal)
3. Şu komutu yaz ve Enter'a bas:

```
npm install
```

4. Yükleme bitince (1-2 dk):

```
npm run dev
```

5. Tarayıcıda aç: **http://localhost:3000**

---

## Her gün çalıştırmak için

Sadece 1 komut yeterli:
```

```

Sonra tarayıcıda: http://localhost:3000
